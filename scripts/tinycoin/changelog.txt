New features added in version 1.7:
	1. Web GUI for invoking API calls
	2. Version API
	3. New fields added to transaction JSON
		timestamp - time of invoking transaction, typically system date
		txnid - generated by suffixing the increment of previous txnid to miner-address
		signature - Exploring the signature inclusion, currently a simple text fields
	4. Fields modified in transaction JSON
		amount - accepts float values also.
Planned features in 1.7:
	1. Populate updated values into appropriate places eg. update miner should display the new value in miner box.
	2. Detect peers calls connect_to_peers_of_peers and return the full peer list.
